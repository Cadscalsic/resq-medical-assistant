# ResQ - Emergency Decision Support System

Save Lives. Anytime. Anywhere.

## Overview
ResQ is a rule-based emergency assistance system designed to provide immediate first-aid guidance without relying on external AI services.

The system uses structured medical logic and predefined decision trees to analyze user input and determine the appropriate response.

## Key Features
- Deterministic decision engine (no AI hallucination)
- Works offline (no internet required)
- Multi-language support (EN, FR, AR, ES, SW)
- Real-time emergency guidance
- Safe and predictable outputs

## How It Works
The application processes user input through:
1. Symptom classification
2. Severity evaluation
3. Predefined medical decision trees
4. Step-by-step guidance output

## Tech Stack
- React + TypeScript
- TailwindCSS
- Custom Rule-Based Engine

## Run Locally
npm install
npm run dev

## Vision
Provide reliable, offline-first emergency assistance accessible to everyone, especially in low-connectivity regions.
